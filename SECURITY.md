# Security policy

## Supported versions

Only the current `main` branch is supported during early development.

## Reporting a vulnerability

Do not open a public issue for a suspected vulnerability or exposed secret.
Use GitHub's private vulnerability reporting for this repository when available,
or contact the repository owner privately.

Include the affected revision, reproduction steps, impact, and any suggested
mitigation. Please avoid accessing data that does not belong to you.

## Local data

Relay is intended for personal research. Local databases, imported documents,
credentials, paid research, and generated analysis must remain untracked. The
repository ignores these paths by default; verify staged changes before every
commit.
